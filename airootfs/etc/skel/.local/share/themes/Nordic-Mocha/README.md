# Nordic-Mocha

A Nordic-inspired Openbox theme with Catppuccin Mocha colors.

Added to ArchBang for Labwc versionm, still WIP

## Based on:
- [Nordic-Openbox](https://github.com/hsully03/Nordic-Openbox)
- [Catppuccin Mocha](https://github.com/catppuccin/catppuccin)

## Features:
- Clean, minimalist button design
- Mocha color palette (dark backgrounds, cyan accents)
- Colored buttons for different window actions:
  - Cyan: Shade button
  - Peach: Desk button
  - Green: Iconify and Maximize buttons
  - Red: Close button
